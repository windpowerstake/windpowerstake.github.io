Terms & Concepts

  • *Total Bytes Written: This node contains a juno|huahua copy. The HDD is monitored for write operations in order to learn the MTBF with TBW as the time passing variable.
  • ‚úù: Particular component, serialized by us. Details about each part will follow.
  • 16h: 16h aggregates (~10000 Blocks)
  • ToDo: I/O write errors. This will be helpful to know when we need to discard the SSD. This should be zero always and the general recommendation is not running anymore the SSD in a node where the first IO writing failure has happened, therefore this detector will be a close call to the end of life of the SSD in order to trigger a replacement.
  • Œ£ : Cumulative sum during the entire life of the applicable component.
