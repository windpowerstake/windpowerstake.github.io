Terms & Concepts

  • *Total Bytes Written: The HDD is monitored for write operations in order to learn the MTBF with TBW as the time passing variable.
  • ✝: Particular component, serialized by us. Details about each part will follow.
  • 16h: 16h aggregates (~10000 Blocks)
  • Inst: Instantaneous measurement at timestamp, 1-off.
  • ToDo: I/O write errors. This will be helpful to know when we might need to discard the SSD. This should be kept at zero and the general recommendation is not to continue running the SSD in a node where the first IO writing failure occured; therefore this detector will be a good indicator of the end of life of the SSD in order to trigger a replacement.
  • Σ : Cumulative sum during the entire life of the applicable component.
  • 🔧: Node under planned maintenance. (if any)
  • 💥: Unplanned maintenance. (if any)
  • ⚡: Power outtage (if any), nodes are covered by an UPS, and now all the fiber routers are also covered by UPS. We would be ready to cover power outtages and/or switch node safely if necessary.
